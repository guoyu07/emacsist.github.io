Percona的MySQL维护及监控工具使用

2016.04.25 emacsist mysql

安装

Doc

使用

pt-mysql-summary

pt-mysql-summary --host=127.0.0.1 --port=6606 --password=xxx

说明： 显示MySQL服务器的概况.

pt-align

将输出的文本对齐.

例子:

pt-archiver

将mysql数据库中表的记录归档到另外一个表或者文件.

默认会删除源表的数据.

pt-config-diff

配置文件比较工具

pt-config-diff my.conf1 my.conf2 ...

pt-deadlock-logger

死锁检测输出日志工具，这里将它输出到test数据库中的deadlocks表中.

pt-diskstats

打印磁盘io统计信息

pt-duplicate-key-checker

重复索引检测工具.

pt-fifo-split

可以模拟切割文件，并通过管道传递给先入先出队列而不用真正的切割文件

pt-find

对MySQL的数据库或表进行类似GNU find的操作并执行一些命令

官方文档

pt-fingerprint

将查询转化为带参查询

pt-fk-error-logger

提取和记录mysql外键错误信息.

pt-heartbeat

监控MySQL复制延迟.

官方文档

pt-index-usage

从慢查询中分析索引使用情况

pt-online-schema-change

在线更改表结构，既不阻塞read，也不会阻塞写

官方文档

pt-query-digest

从日志，processlist, tcpdump分析MySQL的查询.

官方文档

pt-show-grants

显示服务器的用户及相应的权限

pt-sift

浏览由 pt-stalk 生成的文件

pt-stalk

问题诊断工具.

当MySQL的参数 threads_connected 达到 2500 时， 就收集它的服务器状态信息.

pt-table-checksum

用于检测MySQL主、从库的数据是否一致

博客资料

pt-table-sync

同于主从同步.

pt-table-checksum 发现不一致时，就用它来进行同步.

pt-variable-advisor

查看变量及其提供建议.

pt-visual-explain

可视化explain
