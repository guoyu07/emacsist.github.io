结论

alter table xx add column如果不加default默认值，则PG是最快的。只需要在字典里添加一条数据，而对MySQL来说无论有没有，它都要重写整个表及数据.

不过，即使有not null default，PostgreSQL相对于MySQL来说，也是很快的.
