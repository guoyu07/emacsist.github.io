Tomcat 8 源码学习一之导入到IDEA

2016.06.27 emacsist Tomcat源码 ( 9 ) java ( 116 ) source ( 1 ) tomcat ( 21 )

导入Tomcat源码

这里使用的Tomcat的版本为apache-tomcat-8.0.36，源码目录在/ihome/java/tomcat/apache-tomcat-8.0.36-src。在这个根目录下，创建一pom.xml文件，内容如下：

然后删除 test/util/TestCookieFilter.java 这个文件。

最后，在IDEA中选择File -> New -> Project from Existing Sources...，然后定位到/ihome/java/tomcat/apache-tomcat-8.0.36-src即可.

运行

选择Run -> Edit Configurations.

Main class为org.apache.catalina.startup.Bootstrap

VM options为-Dcatalina.home=/ihome/java/tomcat/apache-tomcat-8.0.36-src -Dcatalina.base=/ihome/java/tomcat/apache-tomcat-8.0.36-src -Djava.endorsed.dirs=/ihome/java/tomcat/apache-tomcat-8.0.36-src/endorsed -Djava.io.tmpdir=/ihome/java/tomcat/apache-tomcat-8.0.36-src/temp -Djava.util.logging.manager=org.apache.juli.ClassLoaderLogManager -Djava.util.logging.config.file=/ihome/java/tomcat/apache-tomcat-8.0.36-src/conf/logging.properties

Working directory为：/ihome/java/tomcat/apache-tomcat-8.0.36-src

最后apply-> OK即可。

然后点击Run就可以了。

Github

如果大家不想这么麻烦，我已经将整理后的结构放到了 Github，你也可以直接clone下来，然后直接导致入IDE即可.
