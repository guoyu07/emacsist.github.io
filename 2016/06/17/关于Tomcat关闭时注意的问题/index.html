Task

关闭方式：

shutdown

它会等待线程执行完毕程序再退出，即执行到了”out task ok xxx”

kill

没有执行到out task ok xxx程序就退出了

listener 层

结论

最好停止Tomcat的方式是: 通过Nginx加负载均衡，在Nginx上停用某个Tomcat的连接（upstream)，然后过一段时间后，再调用Tomcat的shutdown.sh，以尽可能减少数据的丢失。
其他的方式来停止Tomcat，都会有风险。
