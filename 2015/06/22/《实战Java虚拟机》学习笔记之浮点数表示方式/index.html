IEEE 754 标准

Java虚拟机处理浮点数的标准是参考 IEEE 754 规范的。

  • 参考文档 : IEEE 754

标示格式

阶码（指数位）

指数部分即使用所谓的偏正值形式表示，实际值为表示值与一个固定值(32位的情况是127)的和

单精度的指数部分是-126～+127加上127 ，指数值的大小从1～254（0和255是特殊值）
